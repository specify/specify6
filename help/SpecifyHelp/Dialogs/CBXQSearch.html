Query Combo Box Search

This dialog allows fields in the table to be searched and a record chosen from the search results.

This search is not inclusive of criteria, it treats all criteria independently. For example; when searching for an agent 'John Smith', entering 'John' into the first name field and 'Smith' into the last name field will return results for all last names of 'Smith' and all first names of 'John'. The search will not be limited to only 'John Smith'.
