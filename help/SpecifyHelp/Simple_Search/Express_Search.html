Simple Search

Specific fields in Specify 6 are indexed to allow quick, ordered searches, called Simple Search. Simple Search allows up to two words to be searched in the indexed fields. Simple refers to the amount of resources and time that it takes to complete a search. Simple Search can be configured for the exact fields needed for a search. Thus, the smaller the number of fields searched, the more 'express' the search is.

Simple Search can also be configured to show results from indexed fields in other tables, or Related Searches. For example; a search for an Agent name can display the name in the Agent table, as well as all Determinations owned by that agent in the Determinations table. At this time the Related Searches are built into Specify and can be activated or deactivated.

To configure a search by choosing from all the fields in Specify 6 use the Query tool. Please refer to New Query.

Using Simple Search

Simple Search is located in the upper right corner of Specify 6. Click the to display a menu of available tables. For information about adding tables to the menu refer to Configure Simple Search.

To execute an Simple Search:

  1. Choose either All or a specific table from the drop down menu. All will search all the tables that have been configured in Simple Search Config. Choosing one table from the list will search that table first and then all related tables.
  2. Type in either one or two words to search
  3. Press the Return key
