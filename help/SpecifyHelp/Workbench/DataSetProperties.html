Data Set Properties

The name and description of a Data Set are contained in the Data Set Properties.

To view and/or edit the properties of a Data Set, right click on a Data Set name on the Sidebar. A context menu will appear:

 

Click 'Edit Properties' to display the Data Set Properties dialog:

Data Set Properties

 

If a Data Set was created from an imported file, the path to the file will display in the 'Source File' area of the dialog.
